# TrulyFreeOCR — Testing Plan

## Philosophy

Test-driven development: write the test first, then implement until it passes.
Each component has a corresponding test class written before the implementation.

## Framework

- **JUnit 5 (Jupiter)** — via Gradle `test` task
- **AssertJ** — fluent assertions (optional, added to `build.gradle.kts` if needed)
- **Test PDFs** — generated by `TestPdfGenerator.java` (Gradle task: `generateTestPdfs`) in `tests/test-files/`

## Test Structure

```
src/test/java/com/trulyfreeocr/
├── pipeline/
│   ├── PageExtractorTest.java
│   ├── ImageSegmenterTest.java
│   ├── JBIG2CompressorTest.java
│   ├── OCREngineTest.java
│   ├── PDFAssemblerTest.java
│   ├── MetadataPreserverTest.java
│   └── PipelineOrchestratorTest.java
└── util/
    ├── PlatformUtilsTest.java
    └── SubprocessRunnerTest.java
```

## Test Categories

### Unit Tests
Test a single class in isolation. Mock external dependencies (native binaries,
file I/O). Fast, no native deps needed.

### Integration Tests
Test a component with its real dependencies (Tess4J, jbig2enc binary, PDFBox).
Marked with `@Tag("integration")`. Run separately from unit tests.

### End-to-End Tests
Test the full pipeline end-to-end on real test PDFs.
Marked with `@Tag("e2e")`. Run only after all units pass.

## Test PDFs

Generated by `TestPdfGenerator.java`. Located in `tests/test-files/`. Regenerate with `./gradlew generateTestPdfs`.

| File | Pages | Description | Primary Tests Targeting It |
|---|---|---|---|
| `blank.pdf` | 1 | All-white page, no content | PageExtractor, ImageSegmenter |
| `simple-text.pdf` | 1 | Single-column clean text | OCREngine, PDFAssembler |
| `multi-page.pdf` | 3 | Pages with different content | PipelineOrchestrator, PageExtractor |
| `two-column.pdf` | 1 | Two-column layout | ImageSegmenter (segmentation) |
| `with-annotations.pdf` | 3 | Bookmarks + text/underline annotations | MetadataPreserver |
| `noisy-scan.pdf` | 1 | Speckle noise + lighting gradient | ImageSegmenter (preprocessing) |

## TDD Workflow Per Phase

### Phase 1 — Foundation (CLI + scaffold)
1. `TrulyFreeOCRTest` — parse `--help`, `--version`, basic arg parsing via picocli
2. `PlatformUtilsTest` — OS detection returns expected values
3. `SubprocessRunnerTest` — runs a known command, checks stdout/stderr/exit code

### Phase 2 — PageExtractor
1. **Write**: `PageExtractorTest`
   - `blank.pdf` → 1 page, dimensions match
   - `multi-page.pdf` → 3 pages, each returns a non-null `BufferedImage`
   - Invalid path → throws descriptive exception
2. **Implement**: `PageExtractor.java`

### Phase 3 — OCREngine
1. **Write**: `OCREngineTest`
   - `simple-text.pdf` → returns `PageResult` with word-level `TextBlock` list
   - Known word "quick" appears in results
   - `blank.pdf` → empty result
2. **Implement**: `OCREngine.java`

### Phase 4 — ImageSegmenter
1. **Write**: `ImageSegmenterTest`
   - Takes `BufferedImage` → returns `SegmentedImage` with mask + bg
   - `two-column.pdf` → mask separates both columns
   - `noisy-scan.pdf` → denoised output
2. **Implement**: `ImageSegmenter.java`

### Phase 5 — PDFAssembler (basic)
1. **Write**: `PDFAssemblerTest`
   - Reconstructs PDF from components
   - Output PDF is searchable (text can be extracted)
   - Output PDF has same dimensions as input
2. **Implement**: `PDFAssembler.java`

### Phase 6 — JBIG2Compressor
1. **Write**: `JBIG2CompressorTest`
   - Falls back gracefully when jbig2enc is not on PATH
   - Compresses a known binary mask
2. **Implement**: `JBIG2Compressor.java`

### Phase 7 — PDFAssembler (JBIG2)
1. **Extend**: `PDFAssemblerTest`
   - Output with JBIG2 is smaller than uncompressed version
   - Visual fidelity preserved

### Phase 8 — MetadataPreserver
1. **Write**: `MetadataPreserverTest`
   - `with-annotations.pdf` → bookmarks preserved in output
   - Annotations count matches
2. **Implement**: `MetadataPreserver.java`

### Phase 9 — PipelineOrchestrator
1. **Write**: `PipelineOrchestratorTest` (integration)
   - End-to-end on `simple-text.pdf` → produces searchable output
   - Output file exists and is valid PDF
2. **Implement**: `PipelineOrchestrator.java`

## Running Tests

```bash
# All unit tests (fast)
$TFOCR_JAVA_HOME/bin/java -jar gradle/wrapper/gradle-wrapper.jar test

# Integration tests only
$TFOCR_JAVA_HOME/bin/java -jar gradle/wrapper/gradle-wrapper.jar test --tests "*IT"

# A single test class
$TFOCR_JAVA_HOME/bin/java -jar gradle/wrapper/gradle-wrapper.jar test --tests "*PageExtractorTest"
```

## Adding New Test PDFs

1. Add a new function in `src/test/java/com/trulyfreeocr/TestPdfGenerator.java`
2. Run `./gradlew generateTestPdfs` to regenerate
3. Reference the file from your test class by path:
   ```java
   var pdf = new File("tests/test-files/my-new-file.pdf");
   ```
